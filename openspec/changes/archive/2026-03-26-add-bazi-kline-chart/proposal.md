## Why

The user wants to showcase the "八字K線圖" (Bazi K-Line Chart) project in their bento-style portfolio. This project is a unique visualization of Chinese Bazi (Four Pillars of Destiny) using a stock-market style K-line chart, which aligns with the portfolio's tech-forward and analytical theme.

## What Changes

- Add a new entry to `data/bento-links.json` for the Bazi K-Line project.
- The entry will include a link to `https://bazi.david888.com`, a title, and a high-quality screenshot/image.
- Potentially add a new card type or styling to highlight this specific project if a standard link card is insufficient.

## Capabilities

### New Capabilities
- `bazi-kline-section`: A new interactive or static section in the bento grid dedicated to the Bazi K-Line project.

## Impact

- `data/bento-links.json`: Will be updated with the new project data.
- `components/BentoGrid.tsx`: May be updated if a new card type is introduced.
- `public/assets/`: New image assets for the project.
